# 					    **Qt翻翻乐----说明文档**

-------------------

​	功能描述：翻翻乐小游戏一共设置20个关卡，每个关卡内有4 * 4 的硬币矩阵，通过点击硬币使硬币翻转，并且连带该硬币四周的硬币一块翻转，当整个矩阵内所有硬币均已翻转为游戏胜利。

----------------

​													整个项目的实现思路概括为以下几个大致的章节：

​																		**一：**开始按钮的设计

​																		**二：**动画效果和动作监听的设计

​																		**三：**选择关卡场景设计

​																		**四：**游戏场景设计

​																		**五：**硬币的封装

-----------
